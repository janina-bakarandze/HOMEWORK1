# Homework 1 — HTML Fundamentals

This repository contains my first homework for learning HTML.  
In this task I practiced the most important beginner-level HTML tags and structure.

 What I practiced
Text & Structure Tags
- `<h1>`–`<h6>` — headings  
- `<p>` — paragraph  
- `<b>` / `<strong>` — bold text  
- `<i>` / `<em>` — italic text  
- `<hr>` — horizontal line separator  
- `<br>` — line break  

 Lists
- `<ul>` — unordered list  
- `<ol>` — ordered list  
- `<dl>` — description list  
- `<dt>` — term in description list  
- `<dd>` — definition in description list  

Layout & Grouping
- `<div>` — block-level container  

 Links
- `<a href="">` — hyperlink  


## What I learned
- How to structure an HTML page  
- How different text tags change meaning and styling  
- How to create lists for organizing information  
- How hyperlinks work and how to add them
